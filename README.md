## project: does error type predict quitting?

**Annie Johansson, Maarten van der Velde, Abe Hofman**\
**2026**

this will be a replication and extension of the quitting effect (seq. errors -> quitting), interacted with error type, and explored across math garden and memory lab datasets. 


### analysis plan

model the probability of a state transition, given: 

 -  state (persisting, soft-quit, hard-quit)  
 -  error (cognitive, non-cognitive) * sequence (1, 2, 3, >3)  
 -  person-covariates (age, ability)  
 -  context-specific covariates (home/school, domain)  
 -  surprise (delta): to what extent was the correctness of a response in line with the prediction from the system?  

### data

I start by exploring (a training subset of) multiplication data from 2023 to 2025.  

### questions 

 -  if we add cognitive and non-cognitive error definitions, markov model becomes hidden?  
 -  correct definition of delta?
